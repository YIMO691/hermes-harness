# ET6 GM 测试完整指南

> 用途：ET6 测试题中用 GM 面板组合测试任意模块
> 覆盖：GM入口矩阵、常用物品ID、测试组合模式、边界测试技巧

---

## GM 面板入口矩阵

GM 主面板是九宫格布局：

| 入口 | 功能 | 可用性 | 走服务端 |
|:---|:---|:--:|:--:|
| **数值** | 改钻石/经验/代金券/标记 | ✅ 完全可用 | 是 |
| **属性** | 查看所有系统属性（含神经系统分类） | ✅ 只读 | 否 |
| **OSS** | 运营数据 | ✅ | 是 |
| **背包** | 查看背包物品 | ✅ 只读 | 否 |
| **境界** | 改境界时期+升级 | ✅ 完全可用 | 是 |
| **DNA** | DNA解锁/升级/升星 | ✅ 完全可用 | 是 |
| **Organ** | 器官相关 | ✅ | 是 |
| **日期** | 改服务器时间（影响限时活动） | ✅ 完全可用 | 是 |
| **系统测试** | 空壳按钮（代码注释掉了） | ❌ 不可用 | — |

⚠️ **UIGMTest 的 Test1~Test5 按钮全是空壳** — 代码注释掉了，点击要么报错要么无响应。不要用。

## GM 账号要求

服务端 GM 命令需要 `player.m_bGm == true`。检查方法：数值 GM → `用户是否是` 字段，值为 `1` 即为 GM 账号。

## 常用物品 ID

| ID | 物品名 | 用途 |
|:--|:---|:---|
| 102 | 神经解锁材料（活性酶类） | 神经模块解锁消耗 |
| 101 | 神经升级材料 | 神经节点升级消耗 |
| 1011 | 钻石 | 抽奖消耗（200/抽） |

## 测试组合模式

### 模式1：测试神经模块

```
1. 境界 GM → 升到够高的境界（满足解锁条件）
2. 添加物资 GM → ID=102 数量=999（解锁材料）
3. 添加物资 GM → ID=101 数量=999（升级材料）
4. 进游戏主界面 → 找神经入口
5. 解锁 → 升级 → 突破
6. 属性 GM → 切到「神经系统」分类 → 验证属性变化
```

### 模式2：测试抽奖模块

```
1. 数值 GM → 普通钻石 → 20000
2. 进游戏主界面 → 找抽奖/探宝入口
3. 单抽 → 看扣钻和结果弹窗
4. 十连抽 → 看10个结果和保底
5. 奖池预览 → 看物品列表和概率
```

### 模式3：测试边界（材料不足）

```
1. 消耗掉所有相关材料（或用 GM 加0个）
2. 点操作 → 应该弹出材料不足/来源提示
3. 验证提示窗存在且不闪退
```

### 模式4：验证属性计算

```
1. 记录当前属性值（属性 GM → 对应系统分类）
2. 执行操作（解锁/升级/突破）
3. 再次查看属性 → 对比差值
4. 手动按公式验算
```

## 边界测试 checklist

| 测试项 | GM操作 | 验证点 |
|:---|:---|:---|
| 材料不足→来源弹窗 | 消耗材料到0 | 弹窗不闪退 |
| 钻石不足→充值提示 | 钻石→0 | 提示"请及时充值" |
| 节点点击→属性动画 | 强化界面点节点 | 气泡动画 |
| 圆满倍率 | 突破到圆满前后对比属性 | 公式(100%+X)×属性 |
| 境界过滤 | 改境界看列表变化 | ShowRank过滤正确 |
| 奖池预览 | 点预览按钮 | 概率+品质排序 |
| 成品基因动画 | 抽到成品基因 | 特殊发光效果 |

## 服务器启动检查

服务端正常启动的标志（控制台输出）：
```
2026-06-21 15:12:45.0102 配置表加载完毕
2026-06-21 15:13:34.7725   生命数值： 30
```

如果 GM 按钮点完无响应，先检查：
1. 服务端是否在跑（看控制台窗口）
2. Unity Console 有没有红色网络错误
3. 账号是否 GM（数值页「用户是否是」=1）
